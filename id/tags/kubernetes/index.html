Cara Konfigurasi High Availabilty Kubernetes Cluster di Ubuntu 20.4

Pada kesempatan kali ini saya akan membagikan bagaimana cara konfigurasi High Availability Cluster Kubernetes di Ubuntu menggunakan HAProxy dan Keepalived. Kita menggunakan Keepalived dan HAproxy untuk load balancing dan high availabilty-nya. Hal yang perlu di persiapkan antara lain: Kebutuhan hosts konfigurasi HAproxy dan Keepalived Konfigurasi clustering Kubernetes menggunakan Kubeadm. Architecture Clusters.
